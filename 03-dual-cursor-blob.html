Dual Cursor Effect

A small dot tracks the mouse tightly while a large soft blob trails behind it. The dot uses mix-blend-mode: difference, the blob uses screen — watch how they interact with dark and light backgrounds.

Light Surface

The dot inverts to black on light backgrounds. The blob's screen blend makes it glow against dark areas and disappear on light ones.

Dark Surface

On dark backgrounds the blob's purple gradient becomes visible, creating a soft ambient glow around the cursor.

Effect: Dual Cursor + Trailing Blob
Source: QClay Design
How: Two fixed-position elements track the mouse via lerp. The dot (fast, 0.1) uses mix-blend-mode: difference. The blob (slow, 0.04) uses mix-blend-mode: screen with concentric circles. Hover targets scale the dot via CSS transitions.
