06 — Infinite Marquee · pause-on-hover
Trusted by · 12 cities
Effect: Infinite marquee w/ pause-on-hover
Mechanism: Duplicate the item set inside the track. Animate transform: translateX(0 → -50%) with a linear loop — when the first set exits, the second set sits in its place, so the viewer sees a seamless loop. animation-play-state: paused on wrap-hover halts the motion. CSS mask-image fades the loop seam at both edges. No JS animation — CSS only.
